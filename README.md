
# RTKQ Basic CRUD APP




## Run Locally

Clone the project

```bash
  https://github.com/NAW2005/react-rtkq-json-server.git
```

Go to the project directory

```bash
  cd my-project
```
Getting Started
```bash
    npm run dev
    # or
    yarn dev
```
Open http://localhost:5173/ with your browser to see the result.
